AutoTag

This is a quick script that aims to clean up MP3 ID3 tags automatically
Features:
-Renames folders to Artist - Album format
-Renames tracks to Track# - Title format
-Attempts to remove remaster tags from album and song titles
-Allows user to specify genre and year tags on a per-album basis
-Removes non-audio files from folders

MAKE SURE TO BACK UP YOUR AUDIO FOLDER BEFORE USING
