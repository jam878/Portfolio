MP3AutoTag

Somewhat working, does not fix titles yet

Simple program that attempts to streamline the process of MP3 file tagging

Many MP3 files contain are poorly tagged with improper capitalization and incorrect genre tags, so MP3AutoTag fixes these errors and renames files and changes tags accordingly.

To use, install the mp3-tagger and titlecase modules using pip or other preferred installation techniques
Then, place the script in a folder with other folders containing albums to be tagged.

BE WARNED, THIS WILL DELETE ALL NON-AUDIO FILES, BACK UP YOUR FILES!
